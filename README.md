# 🐶 Woof App - Android

## Giới thiệu
Woof App là một ứng dụng Android demo được xây dựng bằng **Jetpack Compose**.  
Ứng dụng hiển thị danh sách các chú chó cùng thông tin cơ bản (tên, tuổi, sở thích). Đây là bài lab giúp luyện tập thiết kế UI hiện đại với Compose.
